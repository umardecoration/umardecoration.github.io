### 4️⃣ إضافة صور جديدة

1. ضع الصور في مجلد `data/image/`
2. في `script.js`، أضف الصور إلى مصفوفة `galleryImages`:

```javascript
const galleryImages = [
    { src: 'data/image/your-image.jpg', alt: 'وصف الصورة' },
    // أضف صورك الجديدة هنا
];
```

### 5️⃣ إضافة فيديوهات جديدة

1. ضع الفيديوهات في مجلد `data/video/`
2. في `script.js`، أضف الفيديوهات إلى مصفوفة `videos`:

```javascript
const videos = [
    { src: 'data/video/your-video.mp4', alt: 'وصف الفيديو' },
    // أضف فيديوهاتك الجديدة هنا
];
```
